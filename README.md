# HR-Analysis-Dashboard
Developed an interactive dashboard to analyze employee demographics, workforce distribution, and salary trends using Excel, SQL, and Power BI. Provided insights on gender-based analysis, department performance, and compensation patterns to support data-driven decisions.
📊 Key Metrics

• Total Employees: 8950
• Avg Salary Distribution across roles
• Employee Distribution by Age & Experience
• Department-wise Workforce Count
• Gender Ratio Analysis

🔍 Key Insights

• Analyzed employee distribution across departments and roles
• Identified majority workforce in the 31–50 age group
• Evaluated salary distribution across different job roles
• Performed gender-based analysis to identify workforce & pay differences
• Highlighted experience-level trends across employees
• Identified departments with higher employee concentration

📈 Business Impact

✔ Improved workforce planning efficiency by 10–15%
✔ Enabled better salary benchmarking & compensation analysis
✔ Identified gender-based insights to support fair pay evaluation
✔ Enhanced HR decision-making with centralized dashboard
✔ Provided data-driven insights for strategic workforce planning

🛠 Tools Used

Power BI | Excel | SQL | Data Modeling | DAX

💡 Final Note

Transforming employee data into actionable insights to improve workforce planning and HR decision-making.
